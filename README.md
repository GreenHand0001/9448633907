# 项目简介

## 核心功能
- 基于 YOLOv8 的茶叶病害检测，支持 8 类茶叶病害识别
- 提供完整源码、7300 张标注数据集、训练好的模型及训练结果
- 支持图片、视频及摄像头进行实时检测
- 支持自定义置信度和 IOU 阈值设置
- 提供批量检测功能，可查看并保存所有检测结果
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持选择显示标签和原图，以及检测模型

## 适用场景
- 茶叶种植病害监测与管理
- 农业科研与教学演示
- 茶叶质量控制系统
- 农业技术培训与推广

## 技术特点
- 基于 Python 3.9 开发，集成 OpenCV、PyQt5、Ultralytics 和 Torch 1.9 等技术栈
- 采用 YOLOv8 目标检测算法，具有高精度和实时性
- 提供图形化界面，操作简便
- 支持多种输入方式（图片、视频、摄像头）
- 包含完整的项目文件、UI 源文件和运行教程
